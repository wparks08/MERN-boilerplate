# MERN-boilerplate

This boilerplate uses the following:

- Express
- Mongoose
- Morgan
- React (via create-react-app)
- React Router Dom

Dev dependencies include:

- ESLint
- Prettier
- Nodemon
- Concurrently

# Running

To start, run:
```shell
npm start
```
The `start` script will launch both the backend and frontend concurrently in dev mode, from your local machine.
